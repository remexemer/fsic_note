## Getting Started

### Dependencies

* mkdocs
* mkdocs-material
* mkdocs-git-revision-date-localized-plugin
* mkdocs-minify-plugin
* mkdocs-exclude

We may install the prerequisites as follows but tweaks may be required.

```shell
pip install -r requirements.txt
```

### Installing

```shell
make docs
```

### Browsing

```shell
firefox public/index.html
```
